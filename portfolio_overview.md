# 🛒 Project 1: Supermarket Sales Analysis

### Objective
Analyze supermarket transaction data to understand sales patterns, product demand, and customer purchasing behavior.

### Key Analysis
- Sales trend analysis
- Product category performance
- Customer spending patterns
- Revenue distribution

### Key Insights
- Electronics generated the highest sales revenue.
- Weekend sales were significantly higher than weekdays.
- Evening hours recorded peak sales activity.

---

# 🎓 Project 2: Student Performance Analysis

### Objective
Evaluate student academic performance and identify factors affecting learning outcomes.

### Key Analysis
- Score distribution analysis
- Pass/fail rate calculation
- Subject performance comparison
- Attendance impact analysis

### Key Insights
- Higher attendance correlates with better academic performance.
- Mathematics shows the highest variation in student scores.
- Students performing well in one subject tend to perform well in others.

# 🌦 Project 3: Weather Data Analysis

### Objective
Analyze weather data to identify environmental trends such as temperature fluctuations, rainfall distribution, and humidity patterns.

### Key Analysis
- Temperature trend analysis
- Rainfall pattern evaluation
- Humidity correlation analysis
- Seasonal weather patterns

### Key Insights
- Rainfall increases with higher humidity levels.
- Temperature values fluctuate across seasonal periods.
- Weather indicators demonstrate environmental variability.

---

# 🦠 Project 4: COVID-19 Data Analysis

### Objective
Analyze COVID-19 pandemic data to understand infection patterns, recovery trends, and mortality rates.

### Key Analysis
- Confirmed case trend analysis
- Recovery and death rate evaluation
- Active case calculation
- Correlation analysis between key indicators

### Key Insights
- Confirmed cases strongly correlate with deaths and recoveries.
- Recovery rates are significantly higher than mortality rates.
- Active cases highlight periods of increased healthcare pressure.

# 📈 Project 5: Stock Market Data Analysis

### Objective
Analyze stock market data to identify price trends, volatility patterns, and relationships between financial indicators.

### Key Analysis
- Stock price trend analysis
- Daily return evaluation
- Trading volume analysis
- Volatility assessment

### Key Insights
- Opening and closing prices show strong correlation.
- High trading volumes coincide with price fluctuations.
- Stock prices exhibit short-term market volatility.
