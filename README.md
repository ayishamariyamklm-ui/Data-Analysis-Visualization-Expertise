- Data Cleaning and Preprocessing  
- Exploratory Data Analysis (EDA)

Each project focuses on a different industry domain and follows a **structured analytical workflow used in professional data teams**.

# 🔍 Analytical Process

All projects follow a consistent professional data analysis pipeline:

### 1️⃣ Data Collection
Datasets were obtained from publicly available sources or simulated datasets for analysis practice.

### 2️⃣ Data Cleaning
The datasets were cleaned by:
- Handling missing values
- Removing duplicate records
- Correcting data types
- Standardizing formats

### 3️⃣ Exploratory Data Analysis (EDA)
EDA was performed to understand data patterns using:
- Descriptive statistics
- Distribution analysis
- Trend analysis

### 4️⃣ Feature Engineering
New features were created to improve analysis and generate deeper insights.

### 5️⃣ Data Visualization
Various visualizations were created using:
- Line charts
- Histograms
- Scatter plots
- Heatmaps

### 6️⃣ Insight Generation
Key findings were extracted from each dataset and translated into **practical insights and recommendations**.
